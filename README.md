# FinanceWise - Student Money Tracker

This is the MVP for the TechSprint Hackathon.

## Prerequisites
You must have **Node.js** and **Git** installed to run this project.

## How to Run
1. Open your terminal in this folder.
2. Run `npm install` to download dependencies (React, Next.js, etc).
3. Run `npm run dev` to start the app.
4. Open `http://localhost:3000` in your browser.

## Features
- **Design**: Glassmorphism & Dark Mode.
- **Inputs**: Text, Link (Auto-extract stub), Voice (Stub).
- **AI**: Gemini Integration (Mocked for MVP demo).
- **Stats**: Visual spending breakdown.
- **Database**: Firebase (Configured in `src/lib/firebase.js`).

## Deployment
To share online, push this code to GitHub and import it into Vercel.
